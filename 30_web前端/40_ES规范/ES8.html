# async 和 await

async 函数：(感觉async = 普通函数中return new promise ！！！)

  1. async 函数的返回值为 promise 对象，

  2. promise 对象的结果由 async 函数执行的返回值决定

    • async函数返回的是一个promise对象（结果对象好像不会处于pending状态？！）

      async function f2() {
    console.log(222)
}
console.log(f2());    //结果为：Promise {<fulfilled>: undefined}

----------
当f2函数中有如下语句时，执行的结果为：
    return '测试数据'          //Promise {<fulfilled>: '测试数据'}
    return new Error('xx')    //语法错误，不能return，需要throw
    throw 'ss'                //Promise {<rejected>: 'ss'}
    throw new Error('123')    //Promise {<rejected>: Error: 123
    return new Promise((resolve,reject)=>{resolve(11)})
        //Promise {<pending>} "fulfilled" 11
        //Promise定义后直接就执行了，不需要调用！！！！！！

-

    • async函数调用then方法拿到返回的数据

      async function f2() {
   console.log(222)
   return '测试数据'
}
f2().then(data=>console.log(data))    //拿到数据
//普通函数直接let data=f2()就能拿到数据了，async还需要then下 ！！！

-

await 表达式：

  1. await 必须写在 async 函数中

  2. await 右侧的表达式一般为 promise 对象

  3. await 返回的是 promise 成功的值

  4. await 的 promise 失败了, 就会抛出异常, 需要通过 try...catch 捕获处理

  • async中嵌套async函数

    async function f2() {
    console.log(222);
    return '测试数据'
}

async function f3(){
    let data = await f2()
    console.log(data);
}
f3()                     //输出结果："测试数据"
console.log(f3())        //Promise {<pending>} "fulfilled" undefined

-

  • try...catch捕获异常

    • 方式一：在普通函数中定义

      let p = new Promise((res, rej) => {
    // res(111)
    rej(222)
})

async function f1() {
    try {
        let res = await p;
        console.log(res);
    } catch (error) {
        console.log(error)
    }
}

# Object.keys

let p = {name:'tom',age:20}
console.log(Object.keys(p))        //['name', 'age']

-

# Object.values

Object.values()方法返回一个给定对象的所有可枚举属性值的数组

let p = {name:'tom',age:20}
console.log(Object.values(p))        //['tom', 20]

-

# Object.entries

Object.entries()方法返回一个给定对象自身可遍历属性 [key,value] 的数组

let p = {name:'tom',age:20}
console.log(Object.entries(p))  //[['name', 'tom'],['age', 20]]

//创建map
const m = new Map(Object.entries(p));
console.log(m.get('name'));
-

# Object.getOwnPropertyDescriptors

该方法返回指定对象所有自身属性的描述对象

const school = {
    name: "尚硅谷",
    cities: ['北京', '上海', '深圳'],
    xueke: ['前端', 'Java', '大数据', '运维']
};

console.log(Object.getOwnPropertyDescriptors(school))     结果为：
    // cities: {value: Array(3), writable: true, enumerable: true, configurable: true}
    // name: {value: '尚硅谷', writable: true, enumerable: true, configurable: true}
    // xueke: {value: Array(4), writable: true, enumerable: true, configurable: true}


-------------是这些可配置对象的信息-----------
const obj = Object.create(null, {        //第一个参数null是原型对象
    name: {
        //设置值
        value: '尚硅谷',
        //属性特性
        writable: true,
        configurable: true,
        enumerable: true
    }
});
console.log(obj,Object.getOwnPropertyDescriptors(obj))
