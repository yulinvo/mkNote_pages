# Node.js

# nodejs环境安装

  1. 下载对应的版本 https://nodejs.org/en/download/

    or: https://nodejs.org/dist/v16.16.0/node-v16.16.0-x64.msi

  2. 双击安装，默认安装路径：C:\Program Files\nodejs\

  3. node -v查看nodejs的版本，npm -v查看npm的版本

# nodejs版本升级

查看当前node版本 node -v

  • 方式一：用n升级nodejs（建议使用此方法）

    # 查看当前node版本
$ node -v

# 清除npm缓存
$ npm cache clean -f

# 全局安装n
$ npm install -g n

# 升级到最新稳定版
$ n stable

# 升级到最新版
$ n latest

# 升级到定制版
$ n v14.6.0

# 切换使用版本
$ n 13.10.0 (ENTER)

# 删除制定版本
$ n rm 13.10.0

# 用指定的版本执行脚本
$ n use 13.10.0 some.js

# 升级完成查看 node版本
$ node -v

  • 方式二：去官网 (opens new window)下载 LTS版本的node，然后安装即可

# nodejs升级后问题

  • 问题一：

    描述：node -v正常，npm -v 提示如下

    C:\Users\yulinvo>npm -v
npm WARN config global `--global`, `--local` are deprecated. Use `--location=global` instead.
8.14.0

    解决方法： 修改C:\Program Files\nodejs文件夹下两个文件：npm、npm.cmd，将prefix -g替换为prefix --location=global

# npm

# npm加速

http://npm.taobao.org和 http://registry.npm.taobao.org 将在 2022.06.30 号正式下线和停止 DNS 解析。切换至新域名:

http://npm.taobao.org 				=> 		http://npmmirror.com
http://registry.npm.taobao.org 		=> 		http://registry.npmmirror.com

安装cnpm

npm install -g cnpm --registry=https://registry.npmmirror.com

  • 可以分解成两步
    1. npm install -g cnpm
    2. cnpm [config] set registry https://registry.npmmirror.com

npm install -g cnpm && cnpm get registry 发现仓库被自动配置成： https://registry.npmmirror.com/

用户配置项好像是存放在文件：C:\Users\yulinvo\.npmrc 中 ？？？

# npm版本升级

  1. 查看一下npm当前版本 npm -v

  2. 将npm更新到最新版本 npm install -g npm

    如果要更新到指定版本(@后跟版本号)，npm -g install npm@8.14.0

    npm -g install npm@8.14.0 升级后会在用户变量路径下存在一份，默认是在全局变量路径下的 

# npm常用命令

# cnpm同步模块

直接通过 sync 命令马上同步一个模块, 只有 cnpm 命令行才有此功能:

$ cnpm sync express

当然, 你可以直接通过 web 方式来同步: /sync/express (opens new window)

$ open https://npmmirror.com/sync/express

# npm安装包默认存储位置

graph LR
win系统-->环境变量-->全局["全局path环境变量(C:\Program Files\nodejs\)，如npm"]
环境变量-->用户["用户path环境变量(C:\Users\yulinvo\AppData\Roaming\npm)，如vue、vuepress、yarn"]

配置npm安装的全局模块所在的路径、以及缓存cache的路径

  1. 查看默认路径

    npm config get prefix //默认路径：C:\Users\yulinvo\AppData\Roaming\npm

    npm config get cache //默认路径：C:\Users\yulinvo\AppData\Local\npm-cache

  2. 修改默认路径（需要先创建node_global、node_cache两个文件夹）

    npm config set prefix "D:\Develop\nodejs\node_global"

    npm config set cache "D:\Develop\nodejs\node_cache"
